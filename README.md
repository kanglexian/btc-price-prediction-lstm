# 比特币价格预测系统 (LSTM + Streamlit)

基于比特币日线行情数据集，从价格时序特征与技术指标出发，完成一个深度学习驱动的量化预测项目。核心目标不是只做价格拟合，而是构建严谨的时序预测 pipeline，输出可解释的模型评估结果与未来行情推演，并提供交互式可视化界面。

## 项目目标
1. 构建多维度量化特征工程，覆盖趋势、动量、波动率、成交量四大类技术指标。
2. 搭建单向 LSTM 深度学习模型，严格遵循时序因果性，避免未来数据泄露。
3. 从误差、残差、特征重要性三个维度完成模型评估与可解释性分析。
4. 实现自定义天数的滚动价格预测，并输出收益率推演与可下载结果。
5. 提供 Streamlit 可视化交互界面，支持参数调整、模型训练与结果一键查看。

## 数据集说明
**数据集**：比特币日线行情数据

核心字段：
- date：交易日期
- open：开盘价
- high：最高价
- low：最低价
- close：收盘价
- Volume BTC：比特币成交量
- Volume USDT：成交额（可选）

衍生特征：
- 价格类：对数价格、对数收益率
- 趋势类：SMA、EMA、MACD
- 动量类：RSI、ROC
- 波动率类：布林带、ATR
- 成交量类：OBV
- 时间类：星期、月份周期编码

## 运行说明
### 1. 安装依赖
推荐使用 uv 管理依赖：
```bash
uv sync
若使用 pip：
bash
运行
python -m pip install -r requirements.txt
2. 准备原始数据
将日线行情 CSV 文件放到以下任一位置：
项目根目录：./day.csv
数据目录：./data/day.csv
说明：
项目默认使用 90% 数据训练、10% 数据测试。
day.csv 与模型权重文件体积较大，不建议提交到 GitHub。
3. 启动可视化界面
bash
运行
uv run streamlit run app.py
执行后浏览器自动打开系统界面，默认地址为 http://localhost:8501。
4. 首次使用流程
在侧边栏确认数据文件路径与本地文件路径一致。
点击「重新训练模型」，等待训练完成，权重将自动保存。
点击「加载模型并预测」，查看历史走势、测试集拟合、残差分析、特征重要性与未来预测。
通过滑块调整未来预测天数，重新生成预测结果并支持导出 CSV。
主要输出
bitcoin_model.weights.h5：训练完成的 LSTM 模型权重
plots/*.png：训练曲线、预测对比、残差分析、特征重要性等全套图表
bitcoin_future_XXdays.csv：未来 N 天预测价格结果，支持一键下载
页面内实时展示：MAE、RMSE、MAPE、R² 四项核心评估指标
分析亮点
严格遵循时序因果性：移除双向 LSTM、按时间划分训练 / 验证 / 测试集，杜绝未来信息泄露。
丰富的量化特征体系：整合 20+ 维技术指标与周期编码，提升模型对行情规律的捕捉能力。
多层正则化防护：BatchNormalization、Dropout、L2 正则与 Huber 损失，降低金融时序过拟合风险。
单步预测严谨评估：测试集基于前一日真实价格还原，无误差累积，真实反映模型单步预测能力。
全流程交互式界面：从数据加载、模型训练到结果推演全程可视化，参数可自由配置。

# 比特币价格预测系统 (LSTM + Streamlit)

基于深度学习 LSTM 模型的比特币价格预测可视化系统，集成技术指标特征工程、模型训练评估、未来行情滚动预测与交互式可视化界面，使用 Streamlit 构建，支持一键启动与本地部署。

## ✨ 功能特性
- **数据预处理与特征工程**：自动清洗行情数据，构建对数收益率、移动均线、MACD、RSI、布林带、ATR、OBV 等 20+ 维量化特征，并对周期特征进行正余弦编码
- **LSTM 深度学习模型**：多层单向 LSTM 网络，加入 BatchNormalization、Dropout 与 L2 正则化，使用 Huber 损失函数适配金融时间序列
- **多维度模型评估**：提供 MAE、RMSE、MAPE、R² 四项核心指标，配套实际值/预测值对比曲线、残差分布分析
- **未来行情滚动预测**：支持 7~90 天自定义天数滚动预测，同步输出价格走势与累计收益率，预测结果支持导出 CSV
- **特征重要性分析**：基于 Permutation 方法计算各特征对模型的贡献度，可视化展示特征权重
- **交互式可视化界面**：基于 Streamlit 构建，侧边栏可自由配置参数、切换训练/预测模式，所有图表支持缩放与悬停查看
